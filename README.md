A simple, user-friendly web interface for converting between Gregorian and Lunar calendar systems.

## Features

- **Bidirectional Conversion**: Gregorian ‚Üî Lunar calendar
- **Multi-language Support**: English, Vietnamese, Traditional/Simplified Chinese
- **Leap Month Handling**: Support for lunar leap months
- **Time Zone Aware**: Accurate conversions across global timezones
- **Responsive Design**: Works on desktop and mobile devices

## Usage

1. Open the HTML file in any modern web browser
2. Choose conversion direction (Gregorian‚ÜíLunar or Lunar‚ÜíGregorian)
3. Enter your date and timezone
4. Click convert to see results

The interface automatically detects your timezone and sets today's date as the default.

## API

This interface uses the [Lunar Calendar API](https://github.com/hnthap/lunar-calendar-api).

## Requirements

- Modern web browser with JavaScript enabled
- Internet connection (for API calls)
